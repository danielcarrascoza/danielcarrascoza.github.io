Overview

  • Built a vehicle identification model for parking structure surveillance to flag vehicles associated with criminal activity — targeting the ~10% of property crimes and 7.3% of violent crimes that occur in parking areas.
  • Implemented a ConvNet with pairwise self-attention blocks inspired by Dosovitskiy et al. (ViT) and Zhao et al. ("Exploring Self-attention for Image Recognition"), adapting the SAN architecture as an alternative to ResNet.
  • Model outperforms a ResNet baseline in early training, though at significantly higher computational cost due to the complexity of the attention mechanism.

Architecture

Model Pipeline — SAN Variant
Input 228×228×3 → Transition Layer → SA Block → × 5 repeats
Final SA Block → Global Avg Pool → Softmax (196 classes)
Transition layers: 1×1 conv + max-pool, expanding channel dim while halving spatial dim
SA blocks: pairwise attention via Hadamard product — α(xᵢ,xⱼ) ⊙ β(xⱼ) — no fixed spatial weights
Optimizer: Adam, lr = 1e-3

Self-Attention Mechanism

  • Each SA block splits the feature map into two streams: Stream 1 computes attention weights α(xᵢ,xⱼ) via a relational function using the Hadamard product; Stream 2 computes a value representation β(xⱼ). The output aggregates both: yᵢ = Σⱼ α(xᵢ,xⱼ) ⊙ β(xⱼ).
  • Unlike convolution, pairwise self-attention is a set operator — permutation and cardinality invariant, with no stationary weights anchored to fixed spatial locations. This allows the model to focus on salient vehicle features regardless of position.
  • Feature aggregation adapts per channel since weight computation does not collapse the channel dimension — enabling richer, more expressive representations than standard ResNet layers.

Dataset & Training

Stanford Cars Dataset

8,144 training / 8,041 test images across 196 car model classes. RGB images preprocessed to 228×228. Annotations from a .mat file mapping images to class indices.

Baseline Comparison

Trained against a ResNet model on the identical dataset and split. The self-attention model showed slightly better early-training accuracy, but required significantly more compute per epoch.

Results & Limitations

  • SA model performed comparably to — and marginally above — ResNet at equivalent training steps, consistent with findings in Zhao et al. where pairwise SA networks match or outperform comparable ResNet baselines.
  • Full convergence was not reached due to GPU memory constraints and the high cost of pairwise attention computation; longer training runs on high-end hardware would likely yield stronger gains.
  • Debugging the SA blocks proved difficult given the complexity of the pairwise functions; a key area for improvement is more modular, inspectable block design to isolate attention weight issues.
